§ 03
Prompt-to-game:
в браузере из промпта
Если задача не «мир-модель», а конкретная HTML5-игра, встраиваемая в клиентский сайт за iframe — есть три практических маршрута, с очень разной экономикой и контролем IP.
I
Rosebud AI
Единственная коммерческая платформа с задокументированным iframe-embed. Phaser 3 или Three.js, Stripe-оплата международными картами.
II
Phaser 4 + Claude Code
Open-source MIT, ваш стек. Папка /skills в репозитории = готовые AI-агентские навыки. Чистый HTML5, полное IP. Стоимость — только токены LLM.
III
SEELE AI
Китайский конкурент Rosebud с Three.js WebGL + экспорт в Unity. Полезен как «мост» к серьёзной 3D-игре, если клиент захочет масштаб.
01
⟡ Primary Recommendation · для вашего стека

Phaser 4
+ Claude Code

Open source → чистый HTML5 → полное владение IP

Официальный Phaser-репозиторий (github.com/phaserjs/phaser) в 2026 содержит папку /skills с AI agent skills для Claude Code, Cursor, Copilot — они скаффолдят полные Phaser-игры из промпта. Русский промпт идёт напрямую в Claude/GPT-5, работает идеально.

MIT-лицензия, выход — чистый HTML5/JS, встраивается в любой клиентский сайт <script> или iframe. Стоимость — только токены LLM ($0.50–$5 на игру). Для вашего multi-agent Claude Code стека — наивысшая маржа + полное владение IP.

MIT license HTML5 embed Русский промпт Требует вашего Claude stack
02
Production-ready SaaS

Rosebud AI

Единственный с задокументированным iframe-embed

Text-to-game с выходом на Phaser 3 (2D) или Three.js (3D), работает в браузере, экспорт в itch.io. Free plan — 20 промптов/неделю без коммерческих прав; Pro ~$15–40/мес даёт 100% коммерческие права.

Stripe-оплата — международные карты и Wise проходят без проблем. Крипту напрямую не принимают. Дефолт для массовых клиентских embed-ов, когда скорость важнее контроля.

iframe embed International card Без крипты Free = no commercial

Astrocade · Ludus AI · Krikey AI · Tencent GiiNEX / GameGen-O

Специализированный NPC-движок с голосом, памятью, lip-sync
