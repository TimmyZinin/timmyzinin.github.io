# elkiperepelki.ru

Liquid Glass редизайн сайта семейной базы отдыха «Ёлкино-Перепёлкино».

**Локация:** д. Кочнево, Кимрский р-н, Тверская обл. 140 км от МКАД.
**Владелец:** Миша Зинин · +7 905 126-76-68 · mik.zinin@yandex.ru
**Дизайн / разработка:** Тим Зинин · timzinin.com

## Стек

Чистый HTML + CSS + ванильный JS. Без фреймворков. Статика.

- `index.html` — единственная страница
- `styles.css` — Liquid Glass + warm palette
- `app.js` — переключатель сезонов, плавный скролл, parallax aurora, отправка формы в WhatsApp
- `img/` — все фотографии, WebP

## Запуск локально

```bash
python3 -m http.server 8966
```

## Деплой

GitHub Pages из `main` ветки, в подкаталог `/elkiperepelki/` на `timzinin.com`.

## Дизайн-rationale

Палитра «вечер на реке» — тёплый угольно-зелёный + охра + золото. Liquid Glass с тёплым теплоощущением, не Apple-холодом. Главное фото — банный чан с паром.

Позиционирование от голосовых владельца:
- «Волга вместо Геленджика» — альтернатива южному курорту для семей с детьми.
- Атмосфера «дачи у родных», а не сетевого resort.
- Главный USP — чан с кинотеатром (его не было на старом сайте).
- Фильтр аудитории — отсутствие молодёжных party-формулировок.
