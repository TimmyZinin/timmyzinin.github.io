# Entrepreneur Quest

Браузерная игра-лид-магнит. Сыграй за Марину — владелицу контент-студии — и выбери, как она проживёт свой год. Каждая сцена = бизнес-ситуация, каждый выбор меняет 4 ресурса: энергия, деньги, время, репутация.

**Играть:** https://timzinin.com/entrepreneur-quest/
