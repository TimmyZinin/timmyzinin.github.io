Оплатить подписку Claude Max $100

Claude Code работает через твой аккаунт на claude.ai. Бесплатный и Pro-тарифы не подойдут — лимиты слишком жёсткие для задач, которые мы будем решать (парсинг, аналитика соцсетей, генерация писем в базы журналистов). Стартуем сразу с Max $100/мес — это минимум, который позволяет работать серьёзно.

  • Открыть claude.ai/upgrade
  • Выбрать план Max · $100/месяц (в Anthropic его иногда называют «Max 5×»)
  • Привязать карту, подтвердить

Почему не Pro за $20 На Pro ты упрёшься в лимиты за 20–30 минут активной работы с Claude Code — он тратит токены не как чат, а как настоящий сотрудник: читает файлы, запускает команды, генерирует большие отчёты. Max $100 даёт кратно больше лимитов и нормальный темп работы. Если через месяц увидишь, что этого мало — апгрейднешься до Max $200. Подписку можно отменить в один клик в настройках в любой момент.
03 Шаг 2

Установить Claude Code в терминал

Claude Code — это CLI, то есть «командная строка». Ты общаешься с Claude не в браузере, а прямо в терминале: он умеет читать файлы на твоём компьютере, писать код, запускать команды, искать информацию. Именно из него мы и будем собирать твой персональный скилл для PR и парсинга.

Ниже выбери свою систему:

Установка на Mac

1. Открыть Terminal

Нажать ⌘ + Space (Spotlight), набрать Terminal, Enter. Откроется чёрное/белое окно — это и есть терминал.

2. Установить Node.js (нужен как движок для Claude Code)

Скопировать и вставить в терминал одной строкой. После вставки — Enter. Терминал спросит пароль от Mac — вводи, символов не будет видно, это нормально.

# Сначала ставим Homebrew (менеджер программ для Mac) /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" # Потом ставим Node.js brew install node

3. Установить Claude Code

npm install -g @anthropic-ai/claude-code

4. Запустить

claude

В первый раз Claude Code откроет браузер и попросит залогиниться — тем же email/паролем, что и на claude.ai. После логина можно закрыть вкладку, вернуться в терминал — там будет приглашение к диалогу.

Установка на Windows

Важно для Windows Claude Code на Windows работает через WSL (Windows Subsystem for Linux) — это «Linux внутри Windows». Звучит страшно, но ставится одной командой. Если пойдёт не так — позвони мне до встречи, сделаем вместе за 5 минут.

1. Открыть PowerShell от администратора

Пуск → набрать PowerShell → правой кнопкой → «Запуск от имени администратора».

2. Установить WSL

wsl --install

После этого обязательно перезагрузить компьютер. После перезагрузки появится окно Ubuntu — нужно будет задать логин и пароль (любые, запиши их).

3. Внутри Ubuntu (WSL) установить Node.js

curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash - sudo apt install -y nodejs

4. Установить Claude Code

npm install -g @anthropic-ai/claude-code

5. Запустить

claude

Установка на Linux

Если у тебя Linux, ты и так знаешь что делать, но на всякий случай:

# Node.js через nvm (или ставь свой любимый способ) curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash nvm install 20 # Claude Code npm install -g @anthropic-ai/claude-code # Запуск claude
Что делать, если что-то сломалось Не пытайся чинить сама — это отнимет полчаса и нервы. Сфотографируй ошибку, пришли мне в Telegram. Я отвечу конкретной командой-исправлением. На встрече мы всё равно пройдёмся и убедимся, что всё работает.

08 FAQ

Вопросы, которые точно возникнут

Что такое «терминал» и почему именно он?

Это чёрное/белое окно, в котором ты общаешься с компьютером текстом. Кажется архаичным, но на деле мощнее любого интерфейса: Claude из терминала видит все твои файлы, может их править, запускать программы, лезть в интернет. В браузере этого сделать нельзя.

Claude будет видеть все мои файлы? Это безопасно?

Claude видит только ту папку, из которой ты его запустила — не весь компьютер. Запустила из папки ~/work/pr-hunter — видит только её. Claude Code ничего не отправляет Anthropic кроме твоих сообщений в чате. Конкретные файлы показываются модели только когда ты явно их открываешь или Claude сам решает их прочитать, и это всегда видно в интерфейсе.

Я сломаю что-нибудь?

Сложно. Claude Code по умолчанию спрашивает разрешение перед каждым изменением файла и перед запуском команд. Всё можно откатить. На встрече покажу как работает «режим безопасности», чтобы ты чувствовала себя спокойно.
