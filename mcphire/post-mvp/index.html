Phase III — Scale & monetize

ML

Goal. pgvector-overlay поверх существующих rule-based matches. Similarity по soft-полям (intent, culture, tone). Улучшает ranking в top-5.

S12.1pgvector install + embed_profiles cron~ 4 ч

Установить pg_extension, generate embeddings (OpenAI ada или local) для 120 soft-fields каждого profile + job.description.
