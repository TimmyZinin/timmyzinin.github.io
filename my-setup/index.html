02 / 20 · WORKFLOW
КОНСТИТУЦИЯ + СПРИНТ

Сначала — правила.
Потом инструменты.

Без дисциплины любая AI-инфраструктура превращается в генератор галлюцинаций. У меня есть конституция и спринт-протокол, которые вшиты в каждую сессию Клода.

КОНСТИТУЦИЯ · 7 ПРАВИЛ

  • Честность — код ≠ фича. Сделано = пользователь увидел
  • Проверяй — потом докладывай — curl, screencapture, Telethon
  • Максимальная автономность — 10 шагов эскалации до Тима
  • Контент = картинка + ссылка + UTM — без картинки не публикуем
  • Координация сессий — handoff, activity_log, session_registry
  • Инфраструктура — после деплоя curl, иначе откатываемся

СПРИНТ · 8 ШАГОВ

PLAN → HITL → IMPLEMENT → TEST → SEC → DEPLOY → VERIFY → NOTIFY

Между TEST и SEC — обязательный Codex code review. Между PLAN и HITL — adversarial review. Деплой жму только после ревью второй парой глаз.

Правила > скорость. Всегда.

ОРКЕСТРАЦИЯ

Master-slave между Claude, MiniMax и Codex.

Я работаю в пяти мозгах сразу. Параллельные агентские сессии крутятся постоянно — каждая со своим контекстом, лимитом, задачей. Оркестрация через agent-bus.

MASTER-СЕССИЯ (Opus 4.7)

  • Держит 1M контекста
  • Планирует, решает архитектуру
  • Запускает slave-агентов через /spawn
  • Собирает результаты через /inbox

SLAVE-СЕССИЯ (MiniMax M2.5)

  • Запускается через mm
  • Отдельный CLAUDE_CONFIG_DIR чтобы не конфликтовать
  • Получает ТЗ от master, пишет код
  • Результат ревьюит master перед мерджем

REVIEWER (OpenAI Codex CLI)

  • Read-only sandbox — не может изменять файлы
  • Два режима: review (ищет баги) и adversarial (атакует архитектуру)
  • Вердикт: APPROVED / NEEDS_CHANGES / REJECTED
  • Для критичного кода — /combo-review (MiniMax + Codex)

КАК ЭТО ВЫГЛЯДИТ

Я запускаю master. Master порождает 3 slave'а: один скрапит, второй пишет тексты, третий верстает HTML. Параллельно Codex ревьюит master-план. Я пью кофе. Через 20 минут всё собрано.

Один человек = один мозг. Пять сессий = пять мозгов одновременно.

INSTAGRAM СТЕК + CTA

Отдельно — про Instagram.

Instagram — самая сложная платформа для автоматизации (Meta активно ловит ботов). У меня тут три слоя.

instagram-cli

By @lupikovoleg. Терминальный CLI + локальный MCP-сервер.

  • Поиск по теме (мультиязычный)
  • Viral detection (freshness filter — today/last N days)
  • Скачивание Reels / Posts / Stories / Highlights
  • Анализ профилей — stats, followers, likers, комменты
  • Экспорт в csv/json

HIKERAPI_KEY + OPENROUTER_API_KEY

ig-mcp

Официальный Graph API MCP (jlbadano fork).

  • get_profile_info, get_media_posts, get_media_insights
  • publish_media — upload фото/видео
  • get_conversations, get_conversation_messages, send_dm (Advanced Access)
  • get_account_pages — связка с Facebook-страницами

FB Developer + Long-lived token
