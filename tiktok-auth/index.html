Авторизовать TikTok для постинга

Открой эту страницу в своём обычном Chrome (Profile 4), где TikTok уже залогинен — повторный вход и SMS не нужны. сессия готова

Authorize TikTok →

Нажми кнопку → на экране TikTok нажми Authorize

Что дальше:

  1. Жми Authorize TikTok выше.
  2. TikTok покажет экран согласия — нажми Authorize / Continue.
  3. Тебя перекинет на страницу с кодом — она сама покажет, что скопировать и прислать Claude.

Ссылка содержит только публичные параметры (client_key + PKCE-challenge), секретов нет. Привязана к одноразовой сессии — если устареет, Claude сгенерит свежую.
