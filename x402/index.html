x402

TL;DR — честный вердикт

x402 — реальный стандарт с бэкерами уровня Visa, Google и AWS, и принимать по нему деньги можно без Coinbase и без KYC. Но органический спрос агентов сегодня крошечный: продажа вызовов агентам даст ~$0. Деньги в этой теме сейчас — B2B-сервис «крипто-биллинг под ключ для отрезанных от Stripe» с живой витриной и жёсткими гейтами.

Два скептика (спрос + исполнение) разобрали все 15 идей. Шесть убиты обоими; девять выжили с оговорками. Судья собрал из них один сфокусированный трек из трёх частей — с бинарными гейтами и капом 3–4 сессии. Он опцион на $1–5k — вспомогательный трек рядом с MCPHire и Zinin×Shturbin, которые остаются носителями North Star. Детали — ниже.

01 · Протокол

Как работает x402 — оплата прямо внутри HTTP

Идея простая: код ответа 402 Payment Required лежал в спецификации HTTP тридцать лет без дела. Coinbase в мае 2025 оживил его: сервер отвечает 402 и говорит «плати столько-то USDC на такой-то адрес», агент подписывает платёж и повторяет запрос. Без аккаунтов, API-ключей и карт. Со 2 апреля 2026 протокол — под нейтральным управлением x402 Foundation при Linux Foundation, в коалиции Google, AWS, Microsoft, Cloudflare, Stripe, Visa, Mastercard, Amex, Shopify, Circle, Solana Foundation.

Запрос без оплатыАгент делает обычный HTTP-запрос к платному API
402 + PAYMENT-REQUIREDСервер возвращает список вариантов оплаты: сеть, токен, сумма, адрес payTo
Подпись офчейнКлиент подписывает EIP-3009 transferWithAuthorization и повторяет запрос с заголовком PAYMENT-SIGNATURE. Газ оплачивает релеер
Verify → Settle → 200Facilitator проверяет подпись, проводит USDC-транзакцию в блокчейне, сервер отдаёт результат
Версия V2 (дек 2025)

Заголовки переименованы: X-PAYMENT → PAYMENT-REQUIRED / PAYMENT-SIGNATURE / PAYMENT-RESPONSE. Старые статьи мая 2025 описывают устаревший формат. SDK обратно совместимы.

Сети и токены

Base, Polygon, Arbitrum, World, Solana, Stellar, Ethereum. USDC/EURC через EIP-3009 (gasless), любой ERC-20 через Permit2. Схемы: exact, upto, batch-settlement.

Комиссии

Протокол — 0%. CDP-facilitator Coinbase: 1000 расчётов/мес бесплатно, дальше $0.001/транзакция. Сторонние и self-hosted фасилитаторы — бесплатно, платишь только газ релеера.

Стек за один вечер

pip install "x402[fastapi]" или @x402/express — middleware ~15–20 строк поверх готового API. Клиенты-агенты платят через @x402/fetch / @x402/axios; готовые интеграции есть под LangChain, CrewAI, Dify, Claude MCP, n8n.

Отдельный слой — x402 Bazaar: каталог платных сервисов с собственным MCP-сервером. Эндпоинт индексируется автоматически после первого успешного платежа, и любой MCP-агент может находить и вызывать платные инструменты без ключей. Библиотека @x402/mcp оборачивает MCP-клиент автоматической оплатой — у Anthropic нативного платёжного слоя для MCP нет, монетизация идёт именно через такие обёртки.

02 · Паспорт РФ и KYC

Можно ли принимать деньги без Coinbase. Да.

Этот вопрос агенты проверили отдельным заходом — для тебя он критичен. Протокол permissionless: payTo — просто адрес в блокчейне, личность продавца остаётся за скобками. Слабое место — выбор фасилитатора:

  • CDP-facilitator (Coinbase) — нужен API-ключ через Coinbase-аккаунт. Для гражданина РФ это зона высокого риска блокировки (Coinbase уже массово блокировал российские кошельки, жёсткий санкционный комплаенс). Плюс встроенный KYT/OFAC-скрининг адресов. Для Solana-mainnet CDP — фактически единственный вариант.
  • OpenFacilitator — open-source, «no signup, no account, no rate limits», Base/Solana/Stacks. Работает без Coinbase вообще.
  • x402-rs (facilitator.x402.rs) — Rust, self-hosted: свой фасилитатор на своём VPS со своим релеер-кошельком. Полный контроль.
  • Sentinel, Cloudflare Monetization Gateway и десятки других в awesome-x402 — экосистема фасилитаторов без KYC уже существует.
Три оговорки. 1) Офф-рамп USDC→фиат лежит за пределами протокола — отдельная задача (P2P/крипто-каналы Грузии, у тебя уже решается в других проектах). 2) 20-й пакет санкций ЕС (апрель 2026) вводит полный запрет на РФ-based крипто-провайдеров — любые клиентские услуги в этой теме только для людей физически вне РФ. 3) Индексация эндпоинта в Bazaar при расчёте через сторонний (non-CDP) фасилитатор остаётся под вопросом — её проверяет тестовый платёж до ставки на органический трафик (шаг 1 плана).
03 · Экосистема

Кто зарабатывает — и где надутый пузырь

Здесь главная развилка ресёрча. Витринные цифры и независимая аналитика расходятся в разы:

Витрина (x402.org)

75.41M транзакций и $24.24M объёма за 30 дней, 94k покупателей, 22k продавцов. Кумулятивно 100M+ транзакций на Base, 35M на Solana. Осенью 2025 — рост >10 000% за неделю.

Независимый разбор (Artemis / Chainalysis / Coindesk)

~50% транзакций — self-dealing и wash-trading. Осенний всплеск раздул мем-коин PING; коммерция осталась в стороне. Реальный объём упал на 77% с пика: $5.15M (ноя 2025) → $1.19M (май 2026). В марте — ~$28k реальных денег в день на всю экосистему.

Кто реально зарабатывает: категории всего две. Машиночитаемые данные для агентов — Zyte, Neynar, CoinGecko, StableEnrich (~$3.1k/мес). И LLM-inference-шлюзы — BlockRun YOPO (~$2.7k/мес, 55+ моделей), Hyperbolic (GPU). Топы ниши зарабатывают единицы тысяч долларов в месяц — таков потолок сегодняшнего органического рынка — именно потолок, без иллюзий о нижней планке.

Рынок уже расставил смежные стандарты по слоям: Google AP2 — авторизация (60+ партнёров), Stripe/OpenAI ACP — чекаут в ChatGPT, Skyfire — identity/compliance (Know Your Agent), L402 — Bitcoin-альтернатива. x402 занял позицию нижнего расчётного рельса, поверх которого работают остальные. Показательно: OpenAI свернул Instant Checkout к марту 2026 — флагманский пример агентской коммерции провалился.

Дыры рынка (из независимой аналитики, без наших фантазий): discovery — агенты теряются среди 43k продавцов; доверие — сигналы репутации в Bazaar примитивны; human-in-the-loop — сервисов нет; RU/CIS — пустая клетка, которую все обходят стороной: из-за санкций это скорее минное поле, чем ниша.

Обходить стороной: любых «x402-токенов» как инвестиций. Мем-коины PING/PENG раздували метрики протокола; даже у аналитического сайта x402scan есть одноимённый спекулятивный тикер. Протокол остаётся платёжным стандартом; спекулятивные тикеры под тем же именем — самозванцы.
04 · Экономика микроплатежей

Почему они проваливались 25 лет — и что изменилось

Классика вопроса — Сабо и Одлыжко: все 25 лет микроплатежи убивали mental transaction costs (когнитивная цена каждого микрорешения) и карточные комиссии ($0.22–0.52 минимум за транзакцию); технологии тут вторичны. Агенты + стейблкоины реально снимают обе технические причины: комиссия — доли цента, решение «платить или нет» принимает агент в рамках бюджета.

Главная преграда остаётся: доверие. Проблема сместилась с платёжной системы на самого агента, тратящего чужие деньги, — только 14% потребителей готовы доверить ИИ размещение заказа. Фрод при этом трансформировался и выжил: security-анализ x402 (arXiv) фиксирует TOCTOU-атаки, free-riding, риски двойной траты; chargeback'а в блокчейне нет.

Для сравнения: из 11 000+ MCP-серверов монетизированы менее 5%, типичный доход $500–3000/мес. А «$0.01 × 50 вызовов в день = $15/мес» — прямая иллюстрация того, почему pay-per-call без большого трафика остаётся игрушкой. Красноречивее всего: к июню 2026 95% стоимости x402-платежей — транзакции выше $1. Рынок сам ушёл от микроплатежей к обычным мелким платежам.

06 · Адверсариал-мясорубка

15 идей → два скептика → что выжило

Три идеатора с разными линзами (быстрый доллар / кирки-и-лопаты / контрариан) выдали 15 идей. Дальше каждый концепт атаковали скептик по спросу («кто конкретно заплатит и где доказательства») и скептик по исполнению (KYC, санкции, ToS, цена фокуса). Итог жёсткий — и это правильно.

Убито обоими скептиками — вычеркнуто из списка продуктов

Gate402 — hosted paywall/facilitator SaaS

Конкурируешь с бесплатным OpenFacilitator и Cloudflare. Чужие анонимные платежи через свою инфраструктуру = комплаенс-поверхность платёжного провайдера для грузинского ИП с РФ-паспортом. Фатально.

Iron Curtain Data — вакансии РФ для западных агентов

Западный HR-tech ушёл с рынка РФ; RU-данные для их комплаенса токсичны. Moat есть, покупателя нет. Остаток влит в общий листинг-эксперимент.

Toll Factory / Monetization Kit

Дубли Integration Studio в трёх обёртках — прямое нарушение приоритета ФОКУС. Слиты в одну сущность.

Audio Toll (TTS+STT), 402-Киоск (pay-per-crawl RU)

STT на CPU ломает синхронный 402-флоу (заплатил и остался без ответа). Pay-per-crawl работает только при рычаге Cloudflare; маркетплейс RU-авторов = санкционная поверхность. Убиты.

07 · Рекомендация

Один сфокусированный трек — и гейты, которые запрещено обсуждать

Трек: «Крипто-биллинг для отрезанных от Stripe»

Studio (деньги) + Kokoro-полигон (витрина) + presale курса (воронка)

Почему эта связка: это единственная ветка, где деньги приходят фиатом B2B-чеком на ИП/TBC от людей — все остальные идеи ждут микроплатежей от агентов, которых как класса покупателей пока нет. Три части делят одну инфраструктуру: один шлюз, один кошелёк, один фасилитатор, один нарратив. И боль «принимать деньги без Stripe/KYC» ты прожил лично — рамка переживёт хайп конкретного протокола.

  1. Шаг 1 (1 сессия). Общий x402-шлюз на Contabo: FastAPI middleware + self-custody кошелёк Base + OpenFacilitator. Kokoro за paywall. Self-settle вторым кошельком, верификация on-chain. Критический тест: индексируется ли эндпоинт в Bazaar без CDP-фасилитатора — если нет, discovery-канал мёртв и оффер меняется.
  2. Шаг 2 (1 сессия). Шаблон-репо обвязки + страница оффера «приём платежей за софт без Stripe и KYC» + список 10 DM-целей (билдеры вне РФ + 2–3 живых B2B-клиента Zinin×Shturbin) + драфты DM и presale-поста. Публикации — только по твоему приказу.
  3. Шаг 3. Разослать, ждать гейтов; любая новая стройка в это время под запретом. Фоном — бесплатный датчик: неделя логов внешних агентских вызовов на mcp.mcphire.com.

ГЕЙТЫ (необсуждаемые): ≥2 предоплаты сетапа → делать Studio · ≥5 предзаказов по $49 → производить курс · иначе после 10 DM + поста — закрыть всю x402-ветку и вернуть 100% фокуса на MCPHire и Zinin×Shturbin.

Честная оговорка: весь трек — опцион на $1–5k разовых плюс небольшой хвост, кап 3–4 Claude-сессии до первого гейта. Путь к $20k MRR по-прежнему несут MCPHire и Zinin×Shturbin.
